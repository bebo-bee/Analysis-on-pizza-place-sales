# Pizza Place Sales Analysis
End-to-end data analysis project exploring sales trends, revenue patterns, and customer order behavior using python.

## Project Overview
This project explores a pizza place sales dataset to uncover sales patterns, customer ordering behavior, and revenue trends.
The analysis was conducted using python for data cleaning, transformation, and exoloratory data analysis (EDA).

## Objectives
- Clean and prepare th dataset for analysis
- Explore sales performance across different pizza categories
- Identify peak ordering periods
- Analyze revenue trends and customer purchasing behavior
- Generate insights to support data-driven decision making

  ## Tools & Technologies
  - Python
  - Pandas
  - NumPy
  - Matplotlib
  - Seaborn
  - Jupyter Notebook

  ## Key Insights
  - Identified top-performing pizza categories contributing the highest revenue
  - Discovered peak ordering hours and busiest sales days
  - Observed sales distribution patterns across pizza sizes and types
  - Generated visualization to highlight trends and performance differences

  ## Project Structure
  - 'data/' - Raw dataset files
  - 'notebooks/' - Data analysis notebooks
  - 'images/' - visualizations
  - 'README.md' - Project documentation
 
  ## Conclusion
  The analysis provides insights into sales trends and customer ordering behavior that can help improve product stratrgy, inventory planning, and revenue optimization.
